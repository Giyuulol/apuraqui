# 🗳️ ApurAqui

Aplicativo mobile desenvolvido para faciitar o acompanhamento das eleições e combater a desinformação.
Projeto acadêmico desenvolido para a disciplina de **Desenvolvimento Mobile** da **UNIPÊ**, ministrada pelo professor Leandro Melo (Turno: Noite).

## ✨ Funcionalidades Planejadas

O escopo do protótipo foca em entregar as seguintes funcionalidades para melhorar a experiência do eleitor:

**Acompanhamento de Resultados:**
    Linha do tempo da apuração em tempo real.
    Sistema de favoritos para estados/cidades.
    Notificações de mudanças no ranking.
**Informações do Candidato:**
    Santinho digital e tela de perfil.
    Comparador de propostas.
**Utilidade no Dia da Votação:**
    Localização do local de votação.
    Checklist de documentos necessários.
    Monitor colaborativo de tempo de fila nos locais de votação.
**Transparência:**
    Central de Fake News e notícias confiáveis.

## 🛠️ ecnologias Utilizadas
**Framework:** Flutter
**Linguagem:** Dart
**Integrações (Mock/APIs):** Planejado o uso de APIs públicas (como as do TSE) e/ou dados locais(JSON) para simulação.
