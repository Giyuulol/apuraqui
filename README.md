# 🗳️ ApurAqui

[cite_start]Aplicativo mobile desenvolvido para faciitar o acompanhamento das eleições e combater a desinformação[cite: 50, 54, 78, 101].

[cite_start]Projeto acadêmico desenvolido para a disciplina de **Desenvolvimento Mobile** da **UNIPÊ**, ministrada pelo professor Leandro Melo (Turno: Noite)[cite: 57, 58, 59, 60].

## ✨ Funcionalidades Planejadas

[cite_start]O escopo do protótipo foca em entregar as seguintes funcionalidades para melhorar a experiência do eleitor[cite: 52, 54, 80]:

* **Acompanhamento de Resultados:**
    * [cite_start]Linha do tempo da apuração em tempo real[cite: 109].
    * [cite_start]Sistema de favoritos para estados/cidades[cite: 109].
    * [cite_start]Notificações de mudanças no ranking[cite: 108].
* **Informações do Candidato:**
    * [cite_start]Santinho digital e tela de perfil[cite: 110].
    * [cite_start]Comparador de propostas[cite: 110].
* **Utilidade no Dia da Votação:**
    * [cite_start]Localização do local de votação[cite: 110].
    * [cite_start]Checklist de documentos necessários[cite: 110].
    * [cite_start]Monitor colaborativo de tempo de fila nos locais de votação[cite: 109].
* **Transparência:**
    * [cite_start]Central de Fake News e notícias confiáveis[cite: 110].

## 🛠️ ecnologias Utilizadas
* [cite_start]**Framework:** Flutter [cite: 111]
* **Linguagem:** Dart
* [cite_start]**Integrações (Mock/APIs):** Planejado o uso de APIs públicas (como as do TSE) e/ou dados locais(JSON) para simulação[cite: 98].
